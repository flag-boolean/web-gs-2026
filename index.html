Доступ к Astra Linux:
user: astra; pass: xxXX1234 (Пользователь astra является локальным администратором)

Доступ к Windows 10 Pro:
user: windows; pass: xxXX1234 (Пользователь windows является локальным администратором)

Доступ к Windows Server:
user: Администратор; pass: xxXX1234

Доменные учетные данные:
ALD: user: admin; pass: xxXX1234
AD: user: admin; pass: xxXX1234

Доступ до Linux машин осуществляется посредством программы Putty по SSH, соединение с Windows машинами настроено по RDP.
Переключатель «Фон» в правом верхнем углу меняет оформление (синее/светлое) для удобства восприятия.
